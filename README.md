> Official Homebrew tap for **RelayCraft** — AI-Native Web Traffic Debugging Tool.

## Install

### One-step install

```bash
brew install relaycraft/tap/relaycraft
```

### Tap first, then install

```bash
brew tap relaycraft/tap
brew install relaycraft
```

## Update

```bash
brew upgrade relaycraft
```

## Uninstall

```bash
brew uninstall relaycraft
brew untap relaycraft/tap
```

## About RelayCraft

RelayCraft is an open-source, AI-native network debugging tool for modern developers.

- 🤖 **AI-Native** — Built-in AI assistant, natural language rules and scripts
- 🔌 **MCP Server** — Connect Claude Desktop, Cursor and more via Model Context Protocol
- 🛡️ **Privacy First** — Fully offline, zero accounts, no telemetry
- 🎨 **Modern UI** — Dark mode, glassmorphism, custom themes
- 🔌 **Plugin Ecosystem** — Built-in plugin store with official plugins

Learn more at [relaycraft.dev](https://relaycraft.dev) or check the [main repo](https://github.com/relaycraft/relaycraft).
